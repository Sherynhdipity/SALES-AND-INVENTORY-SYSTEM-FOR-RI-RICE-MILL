Java Reporting Component SDK

 

Java Reporting Component（JRC）により、レポートの表示やエクスポートの機能を既存の Web アプリケーションに埋め込むことができます。JRC は、一連の Java クラスから成る Java ビューアを提供します。これらの Java クラスを使用して、Crystal Reports を表示およびエクスポートする HTML ページや Dynamic HTML ページを生成できます。たとえば、JRC を使用して、JSP ページに直接埋め込まれる軽量なレポート アプリケーションを作成できます。
