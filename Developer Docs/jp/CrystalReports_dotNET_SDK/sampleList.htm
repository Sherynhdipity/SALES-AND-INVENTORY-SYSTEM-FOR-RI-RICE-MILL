Crystal Reports .NET SDK のサンプル コード

 

Web プロジェクトおよび Windows プロジェクト（Visual Studio .NET 2002/2003 と Visual Studio 2005）

Crystal Reports .NET の複数のサンプル コード プロジェクトが含まれています。各プロジェクトは、対応するチュートリアルの手順に従って作成されています。手順と説明は、開発者ガイドの「チュートリアルとサンプル コード」のセクションにあります。

 

チュートリアルのサンプル コード プロジェクトを実行する場合は、コードに含まれるマシン固有の値をチェックし、使用するマシンに合わせてそれらをカスタマイズしてください。次の値が含まれます。

  • レポートの ODBC データ ソースの設定
  • ファイル ディレクトリ パス
  • ネットワーク プリンタ パス
  • 次に示すデータベース接続の API プロパティ設定
    • データベース サーバー名
    • データベース名
    • ユーザー ID
    • パスワード
    • 完全なデータベース接続文字列
