Crystal Reports .NET SDK

 

Crystal Reports により、GUI ベースのプログラムで複雑かつ高度なレポートを作成できます。このレポートからは、結果セット（ADO.NET DataSet など）などのプロキシ データを含む、ほとんどすべてのデータ ソースに接続できます。GUI デザイナに付属するウィザードを使用すると、データを簡単に書式設定、グループ化、および表示できます。

 

Crystal Reports XI R2 は、Visual Studio .NET 2002/2003 と Visual Studio 2005 開発環境に直接統合されます。Crystal Reports ビューア コントロールの 1 つを使用して Web アプリケーションや Windows アプリケーションでレポートをホストしたり、豊富な API を使用して実行時にプログラムでレポートを操作することができます。
