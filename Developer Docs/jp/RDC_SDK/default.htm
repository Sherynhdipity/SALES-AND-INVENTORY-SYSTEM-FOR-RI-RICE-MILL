Report Designer Component COM SDK

 

Report Designer Component（RDC）は、アプリケーションにレポートの編集機能と表示機能をすばやく組み込むことができる Visual Basic 開発者向けのレポーティング ソリューションです。RDC では、SDK を使ってプログラムでも、ビルトイン ユーザー インターフェイスを使って手動でも、レポートを開いたり設計したりカスタマイズすることができます。
