Crystal Reports .NET SDK Sample Code

 

Web and Windows projects (Visual Studio .NET 2002/2003 and Visual Studio 2005)

Contains multiple sample code projects for Crystal Reports .NET. Each project has been built by following the instructions in their corresponding tutorials, and instructions and explanations have been placed within the Tutorials and Sample Code section of the developer guide.

 

If you wish to run the tutorial sample code projects, check the code for any machine-specific values and confirm that they are customized for your specific machine. These may include:

  • ODBC data source settings for reports.
  • File directory paths.
  • Network printer paths.
  • API property settings for database connections, including:
    • Database server names
    • Database names
    • User IDs
    • Passwords
    • Complete database connection strings
