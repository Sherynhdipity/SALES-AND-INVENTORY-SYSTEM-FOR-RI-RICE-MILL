Report Designer Component COM SDK

 

The Report Designer Component (RDC) is a reporting solution that allows Visual Basic developers to quickly integrate report editing and viewing into their applications. The RDC allows you to open, design, and customize reports both programmatically with the SDK and manually with the built-in user interface.
